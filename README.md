Dieses Repository enthält die Programme, die im Rahmen der Abschlussarbeit „Neuronale Netze zur Lösung elliptischer Optimalsteuerungsprobleme: OSNN-Verfahren im Vergleich zur FEM“ erstellt und verwendet wurden. Ein Teil der Python-Implementierungen basiert auf dem ursprünglichen OSNN-Code der Autoren, der unter [DYC1/OSNN](https://github.com/DYC1/OSNN) veröffentlicht wurde, und wurde für die in dieser Arbeit betrachteten Experimente und Auswertungen weiterentwickelt. Zusätzlich enthält das Repository FEM-basierte Referenzprogramme in MATLAB sowie ergänzende Auswertungs- und Visualisierungsskripte.
